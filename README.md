# ChecklistSLApi

Web service for ChecklistSL project

# Installation 

1. Install project dependencies:

    `npm install`

2. Install nodemon:

    `npm install nodemon -g`

3. Install mongoDB

4. Created `ChecklistSLDb` database

# Launch Api

MongoDB

`mongod`

Dev

`npm run dev`

Prod

`npm start`
